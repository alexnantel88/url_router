
Router
------------------------------------

Version: 0.1
Author: Rob Philp (robert.h.philp@gmail.com)
Build Date: 2010-01-17
Requirements: Symphony 2.0.7


[INSTALLATION]

1. Upload the files into a folder named 'routing' in your Symphony 'extensions' folder.

2. Enable it by selecting the "Rotuing" entry, choose Enable from the with-selected menu, then click Apply.

3. You can now use the Routing plugin.
